## 계산기
Java를 처음 학습하게 됐는데 빨리 Java가 익숙해졌음 좋겠다 싶고 기본구조도 익힐겸 아주 간단하게 계산기를 만들어봤습니다.<br>

### 기능 설명
  1. 사칙연산을 메소드로 구현했습니다.
  2. 자바 표준 라이브러리 Java.util의 Scanner를 사용해서 숫자 두개를 입력받을수 있습니다.

### 느낀점
 * Java는 입력을받을때 왜 Scanner객체를 선언하고 입력을 받을까? 라는 생각을 하게됐습니다. [블로그](https://thundercoding.tistory.com/34)
 * 객체지향을 이해하기 위해서 더 열심히 노력해야겠다고 생각이 들었습니다.
